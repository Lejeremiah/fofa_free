# fofa_free
fofa for free


### step 1:
```
pip install -r requirements.txt
```

### step 2:
Modify `config.yaml.example` to `config.yaml`.
```angular2html
account:
    fofa_token: this is your cookie value.

search:
    content: this is your fofa search grammar.
    page: this is number of pages that you want to search.
```
